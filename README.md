# DocChat — Chat with your PDF

> Ask questions about any PDF and get answers with exact source citations.

## Live Demo
[https://fullstack-rag-chatbot.vercel.app/] — upload any PDF and try it

## What it does
- Upload a PDF
- Ask questions in plain English  
- Get answers with page numbers and source chunks

## Built for
Recruiters screening CVs. HR teams querying policy docs.
Anyone who reads documents for a living.

## Stack
FastAPI · React · ChromaDB · OpenAI Embeddings · Groq LLaMA 3.3
